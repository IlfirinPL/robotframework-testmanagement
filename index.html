Robotframework Test Management Library

Test Management Library contains utilities for Robot Framework to connect to and play with Rally REST API. It allows so far for connection, finding test cases and adding test results (with attachments).

API

  • v0.1 API

Requirements

  • Python / Jython / IronPython
  • Robotframework
  • Rally REST Web Service
  • Rally python toolkit

Install

Using easy_install

easy_install robotframework-testmanagement

Using pip

pip install robotframework-testmanagement
